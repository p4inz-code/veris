# @veris/rules

Shipped rule packs for VERIS analysis.

## Rule Packs

- **secrets** — AWS keys, GitHub tokens, generic API keys, private keys, Slack tokens
- **misconfiguration** — Debug enabled, permissive CORS, world-writable files
- **injection** — SQL injection, command injection, path traversal
- **crypto** — Weak hash algorithms, hardcoded keys
- **file-system** — Unsafe unlink, symlink attacks
- **best-practices** — Eval usage, deprecated API usage
